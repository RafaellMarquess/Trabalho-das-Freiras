# Trabalho das Freiras

Projeto acadêmico de **site institucional** desenvolvido para fins educacionais.

## Descrição
Site estático do **Colégio de Freiras Franciscanas Nossa Senhorinha de Itaipu**, com foco em apresentação institucional, valores, informações gerais e contato.  
O projeto prioriza clareza visual, navegação simples e estrutura organizada.

## Objetivo
- Apresentar a instituição e seus valores cristãos
- Informar visitantes sobre a escola
- Disponibilizar página de contato/matrícula
- Praticar HTML, CSS e organização de projeto web

## Tecnologias Utilizadas
- HTML  
- CSS  

## Páginas
- **Home** (`index.html`)
  - Apresentação institucional
  - Sessão “Sobre”
  - Carrossel de imagens
  - Navegação por âncoras
- **Contato / Matrícula** (`contato.html`)
  - Formulário com dados pessoais
  - Upload de comprovante de residência

## Funcionalidades
- Layout responsivo básico
- Navegação por menu
- Formulário com campos obrigatórios
- Estrutura sem backend (apenas front-end)

## Estrutura do Projeto
- `index/` – páginas HTML
- `style/` – arquivos CSS
- `fotos/` – imagens do site
- `README.md` – documentação do projeto
